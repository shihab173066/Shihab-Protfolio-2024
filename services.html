Student Study Portal

Technologies Used: Python, Django, SQLite, HTML, CSS, JavaScript

A comprehensive student portal... that allows users to register, manage tasks, take notes, read books, watch YouTube tutorials, and perform various conversions. This platform is designed to help students stay organized and maximize their productivity. Users can track academic tasks, maintain notes, explore educational resources, and improve learning efficiency with interactive features.

Social Media Application

Technologies Used: Python, Django, SQLite, HTML, CSS, JavaScript

Social Media Platform.... this platform allows users to create accounts, connect with others, and engage in a dynamic social environment. Users can have multiple profiles, follow other people, like posts, and interact with content shared by others. Each user can upload images, create posts, and share their thoughts with the community. Additionally, users can search for other people and their posts, enabling them to connect with others who share similar interests. The platform is designed to facilitate social interactions, encourage networking, and foster community-building through real-time updates and engaging content.

Task Management Application

Technologies Used: Python, Django, SQLite, HTML, CSS, JavaScript

In Task Management System.... I have implemented a simple Task Management system where users can create, read, update, and delete tasks (CRUD functionality) for their To-Do lists. Users can add tasks with details like title, description, and due date. The system allows users to view tasks based on their status—completed or pending—making it easier to track their progress. Additionally, users can manage multiple tasks and see the overall task count, along with the count of completed and pending tasks for better organization and task prioritization. This platform supports multiple users, allowing each user to maintain their own set of tasks, ensuring privacy and customization. The system provides a user-friendly interface where users can mark tasks as completed or pending, delete tasks that are no longer relevant, and update the status or details of existing tasks.

Music Player

Technologies Used: Python, Django, SQLite, HTML, CSS, JavaScript

Music Player... Single users can upload songs, lyrics, and images to create a personalized playlist. This platform allows users to listen to their favorite tracks from anywhere, providing an interactive and seamless music experience. With the ability to upload custom images and lyrics alongside each song, users can enjoy a fully immersive listening experience. The system offers a smooth playback interface with features such as play, pause, and skip, making it easy for users to navigate through their music library. Whether on desktop or mobile, the music player ensures that users can enjoy their music anytime, anywhere, with the added touch of their own images and lyrics for a truly unique experience.

This project features a user-friendly interface, where customers can easily view their selected items, adjust quantities, and proceed to checkout with the applied discounts.
